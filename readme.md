# 🚢 Titanic Survival Prediction - ML Classification Project

This project uses supervised machine learning models to predict whether a passenger survived the Titanic disaster based on features like age, sex, fare, class, and more.

---

## 📁 Dataset

We use the **Titanic dataset** from [Kaggle](https://www.kaggle.com/datasets/brendan45774/titanic), a well-known binary classification problem.

File used: `Titanic-Dataset.csv` (based on `train.csv` from Kaggle)

---

## 🔍 Project Overview

### 🎯 Goal:
Predict survival (`Survived` column: `0 = No`, `1 = Yes`) using other passenger data.

### 🔢 Features Used:
- Pclass
- Sex
- Age
- SibSp
- Parch
- Fare
- Embarked
